# 🛒 Produtos API
## Demonstração

![API funcionando](docs/scalar.png)

Minha primeira API REST desenvolvida utilizando C# e ASP.NET Core (.NET 10).

> ⚠️ Projeto de estudos. Ainda estou aprendendo conceitos como APIs REST, DTOs, Repository Pattern, tratamento de exceções, injeção de dependência e arquitetura em camadas.

## 🎯 Objetivo

O objetivo deste projeto não é criar uma aplicação pronta para produção, mas sim praticar os fundamentos do desenvolvimento backend com .NET.

Durante o desenvolvimento estou aprendendo:

* Programação Orientada a Objetos (POO)
* APIs REST
* Controllers
* Services
* Repositories
* DTOs
* Tratamento de exceções
* Injeção de Dependência (DI)
* Boas práticas de organização de código

## 🚀 Funcionalidades

* Criar produtos
* Buscar produto por ID
* Listar produtos
* Atualizar produtos
* Remover produtos

## 📚 O que aprendi até agora

Com este projeto consegui entender melhor o fluxo:

```text
Requisição HTTP
        ↓
Controller
        ↓
Service
        ↓
Repository
        ↓
Armazenamento dos dados
        ↓
Resposta HTTP
```

Também comecei a praticar:

* Separação de responsabilidades
* Encapsulamento
* Uso de interfaces
* Criação de exceções personalizadas
* Estruturação de APIs REST
